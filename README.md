# Automate Asia App
This app adds support for Z-wave devices sold by Automate Asia, written in SDK2 but not supported by Aeotec App.

**Supported devices:**   
* Aeotec Heavy Duty Switch   
* Aeotec Water Sensor 6
* Aeotec Door Sensor 5
